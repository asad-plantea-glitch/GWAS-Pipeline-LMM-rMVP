# GWAS-Pipeline-LMM-rMVP
An integrated Genome-Wide Association Study (GWAS) pipeline for complex trait analysis using linear mixed models (LMM), population structure analysis, and SNP association mapping implemented in R.
This workflow estimates genotype BLUEs, broad-sense heritability, population structure (PCA and kinship), and performs GWAS using GLM, MLM, and FarmCPU models through the rMVP package. The pipeline is designed for plant breeding and genomics studies utilizing SNP markers and phenotypic trait data.

**Pipeline Overview**

The pipeline consists of three major components:

Phenotypic analysis

Population genomic analysis

Genome-wide association analysis

**Step 1**: Estimation of BLUEs and Heritability

Script:

scripts/01_BLUEs_heritability_LMM.R

Phenotypic traits are analyzed using linear mixed models implemented in the lme4 package.

Model used for BLUE estimation:

model <- lmer(Trait ~ Genotype + (1|Replication), data = data)

Where:

Genotype = fixed effect

Replication = random effect

BLUEs provide adjusted genotype means accounting for environmental variation.

Heritability Estimation

Variance components are estimated using a mixed model where genotype is treated as a random effect:

model <- lmer(Trait ~ (1|Genotype) + (1|Replication), data = data)

Broad-sense heritability is calculated as:

H² = σ²g / (σ²g + σ²e / r)

Where:

σ²g = genotypic variance

σ²e = residual variance

r = number of replications

Outputs:

Genotype_BLUEs.csv

Trait_Heritability.csv

**Step 2**: Population Structure Analysis

Script:

scripts/02_population_structure_PCA_kinship.R

Population structure is estimated using principal component analysis (PCA) and kinship matrix estimation.

PCA identifies major axes of genetic variation among genotypes.

Outputs:

PCA_3D_Plot.jpg

Population_Structure_Groups.txt

A neighbor-joining tree is constructed using the ape package to visualize genetic relationships among genotypes.

Output:

Neighbor_Joining_Tree_Circular.jpg

**Step 3**: Genome-Wide Association Study (GWAS)

Script:

scripts/03_GWAS_rMVP_pipeline.R

GWAS is conducted using the rMVP R package, which supports efficient analysis of large genomic datasets.

Models implemented:

GLM
MLM
FarmCPU

General GWAS model:

Phenotype = SNP + Population Structure (PCs) + Kinship

The analysis integrates:

SNP genotype matrix

phenotype BLUEs

principal components

kinship matrix

Outputs generated by rMVP:

Manhattan plots

QQ plots

GWAS result tables

Significant SNP markers

Significance Threshold

The Bonferroni correction is typically used to control for multiple testing:

Bonferroni threshold = 0.05 / number_of_SNPs

Example:

0.05 / 3093 = 1.62 × 10⁻⁵
